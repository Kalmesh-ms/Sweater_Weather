# sweater_Weather

Weather App with OpenWeatherAPI, for TOP Project.

## Tech Stack

- HTML
- CSS
- JavaScript
- API


## Features

- Current Location
- Search Location
- Current Weather 
- Sunrise & Sunset
- Air Pollution
- 5 Days Forecast
- Hourly Forecast
- Wind Speed
- Based on UTC Timezone
